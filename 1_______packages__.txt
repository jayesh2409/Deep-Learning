TensorFlow (TF):

TensorFlow is an open-source deep learning framework developed by Google.

It allows efficient training and deployment of neural networks on CPUs, GPUs, and TPUs.

Keras:

Keras is a high-level neural network API that runs on top of TensorFlow.

It is user-friendly and simplifies building, training, and testing deep learning models.

Theano:

Theano is an early deep learning library used for mathematical and GPU-based computations.

It inspired modern frameworks like TensorFlow and Keras but is now discontinued.

PyTorch:

PyTorch is an open-source deep learning framework developed by Facebook’s AI Research lab.

It is popular for its dynamic computation graph and easy debugging, widely used in research and production.
